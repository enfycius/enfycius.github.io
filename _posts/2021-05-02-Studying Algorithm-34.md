# 문제

아래 예제와 같이 고양이를 출력하시오.

## 입력

없음.

## 출력

고양이를 출력한다.

### 예제 입력 1

```shell

```

### 예제 출력 1

```shell
\    /\
 )  ( ')
(  /  )
 \(__)|
```

<br/>

# 코드

```cpp
#include <iostream>

using namespace std;

int main(void) {
    cout<<"\\    /\\"<<'\n';
    cout<<" )  ( ')"<<'\n';
    cout<<"(  /  )"<<'\n';
    cout<<" \\(__)|";

    return 0;
}
```
