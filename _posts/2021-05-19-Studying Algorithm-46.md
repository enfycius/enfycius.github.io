# 문제

마이크로소프트 로고를 예제 출력처럼 출력하는 프로그램을 작성하시오.

## 출력

마이크로소프트 로고를 예제 출력처럼 출력한다.

### 예제 입력 1

```shell
```

### 예제 출력 1

```shell
       _.-;;-._
'-..-'|   ||   |
'-..-'|_.-;;-._|
'-..-'|   ||   |
'-..-'|_.-''-._|
```

<br/>

# 코드

```cpp
#include <iostream>

using namespace std;

int main(void) {

    cout<<"       _.-;;-._"<<endl;
    cout<<"'-..-'|   ||   |"<<endl;
    cout<<"'-..-'|_.-;;-._|"<<endl;
    cout<<"'-..-'|   ||   |"<<endl;
    cout<<"'-..-'|_.-''-._|"<<endl;

    return 0;
}
```
